## 버전 정보

### 프론트

- node 22.13
- React (Vite) 18
- VS Code

### 백엔드

- Spring Boot
- JDK 17
- JAR 이용
- IntelliJ 최신 버전 이용
- 제공된 AWS 서버 사용
- OpenVidu 2.31
- Redis 7.4.2
- MySQL 8.4.4
- 젠킨스

### AI
- FastAPI
- OpenAI (GPT-4o-mini)
- LangChain
